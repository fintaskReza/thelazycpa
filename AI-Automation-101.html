1 What is an API Key?

Think of it like a hotel room key.

The hotel = The app/service (like OpenAI, Twitter, etc.)
Your room = Your account
The key = Your API Key

An API (Application Programming Interface) is just a way for two apps to talk to each other. The API Key proves you're allowed to use the service.

Example: You want your automation to post to Twitter. Twitter needs to know it's YOU, not some random hacker. Your API Key proves your identity.

âœ“ DO

  • Store keys in a safe place
  • Use environment variables
  • Rotate keys if leaked

âœ— DON'T

  • Share keys publicly
  • Post them on GitHub
  • Email them to people

2 What is a Terminal?

It's like texting your computer.

Instead of clicking buttons, you type commands.
It's faster and more powerful once you learn it.

Claude Code lives in the terminal. You type what you want, it does it.

Mac: Open "Terminal" app
Windows: Open "Command Prompt" or "PowerShell"

Common commands:
cd folder-name = go into a folder
ls (Mac) / dir (Windows) = list files
python script.py = run a Python file

Cloud Hosting = Running on someone else's computer (server) that's always on. Everyone can access it.

4 What is JSON?

A way to organize data.

Think of it like a labeled filing cabinet.
Every drawer has a name and contents.

JSON is everywhere. APIs send data in JSON. Config files use JSON. It's just structured text.

5 Array vs Object vs Variable

Three ways to store data.

Variable = One thing
name = "Reza"

Array (List) = Many things in order
apps = ["Todo App", "Calculator", "Chat Bot"]
Get first app: apps[0] = "Todo App"

Object (Dictionary) = Things with labels
user = {"name": "Reza", "age": 30}
Get name: user["name"] = "Reza"

6 What is an .env File?

A secret storage box.

Keeps your API keys and passwords separate from your code.

When you share your code on GitHub, you DON'T share your .env file. It stays secret on your computer.

File: .env
OPENAI_KEY=sk-abc123xyz
TWITTER_KEY=my-secret-key
DATABASE_PASSWORD=super-secret

11 What is a Cron Job?

An alarm clock for your code.

Runs your script automatically at set times.

Examples:
â€¢ Post to Twitter every day at 8am
â€¢ Check emails every 5 minutes
â€¢ Backup your database every Sunday

Cron syntax:
0 8 * * * = Every day at 8:00 AM
*/5 * * * * = Every 5 minutes
0 0 * * 0 = Every Sunday at midnight

12 What is an Auth Token?

A temporary pass.

Like a concert wristband. Proves you already checked in.
Expires after a while for security.

API Key = Permanent (until you revoke it)
Auth Token = Temporary (hours, days, or weeks)

Real life:
â€¢ API Key = Your house key (permanent)
â€¢ Auth Token = A hotel key card (expires after checkout)

âœ“ DO

  • Let tokens expire naturally
  • Refresh them when needed
  • Store in memory temporarily

âœ— DON'T

  • Hard-code tokens
  • Commit tokens to Git
  • Share active tokens
